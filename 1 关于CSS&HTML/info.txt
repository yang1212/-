* 盒模型
    盒模型分为IE盒模型和W3C标准盒模型：
    IE盒模型：width = border + padding + 内容的宽度
    W3C标准盒模型： width = 内容的宽度
    https://www.zhangxinxu.com/wordpress/2016/09/talking-about-css-margin-box/

* 居中布局
    常用水平：
      块元素：margin的auto
      行内元素： text-align:center;
      absolute + transform
      flex + justify-content: center
    常用垂直：
      line-height: height
      absolute + transform
      flex + align-items: center 
    水平垂直居中
      absolute + transform
      flex + justify-content + align-items
