## Git
### 命令行
* 查配置(非全局少个--global而已)：git config -l 
* 查全局配置：git config --global -l 
* 设置代理(全局): git config --global https.proxy http://127.0.0.1:1080
* 取消代理(全局): git config --global --unset http.proxy
